# 간단한 3D 대시보드 프로젝트
## 설명
1. 대시보드 우측 영역에서 위젯을 추가/수정/삭제 할 수 있다.
2. 설정한 대시보드들은 새로고침 전까지는 데이터가 저장되어있다.
3. 날라가지 않도록 로컬스토리에 저장하는 설정도 가능하다.

## 위젯 종류
1. 지구본 (라인, 파티클 사용)
2. 구 (라인, 파티클 사용)
3. 시계
4. 텍스트

## 프로젝트 시작 방법
### `yarn start`
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
